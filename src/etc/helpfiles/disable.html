Disabling an overlay

There are likely to be times when you will wish to remove an overlay from the display temporarily, in order to better see a different overlay, or just to speed up display time for a while. You could do this by simply removing the overlay, but then any changes you have made to it will disappear. A better way is to disable the overlay.

To disable an overlay, move to the global icon in the display window (not in the icon bar), and pull down the menu with the left mouse button. Select Disable/reenable and release the mouse button.

The display will now redraw without the disabled overlay. The overlay's icon will change to a gray color (i.e. an icon like turns into ). Any changes you have made to that overlay will be preserved; simply select the same menu entry again to restore the overlay to the display.
