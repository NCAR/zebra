Displaying coordinates in horizontal plots

There are several different types of coordinate overlays which are available for horizontal plots. They include:
  • Latitude/longitude grids
  • X/y kilometer grids
  • Azimuth lines and range rings
  • Maps
  • Locations of static and moving objects
All of these coordinate overlays are available via the overlays icon in the icon bar. Simply pull down the menu, select the overlay type of interest, then click on the destination window. Once an overlay has been added to a display window, there are no end of parameters that can be changed via the icon in the graphic window, including:
  • Origins (except for lat/lon grids)
  • Grid spacings
  • Color
  • Line widths
  • Labelling
And so on.
