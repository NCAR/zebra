Event Logger

The Event Logger will automatically be displayed whenever Zeb is runnning. It provides feedback, information, and error messages from all of the Zeb processes. If Zeb appears to be operating incorrectly, or not at all, check for messages in the Event Logger. The message window in the Event Logger can be scrolled backward and forward over old messages using the scroll bar on the left side of the window.

The button erases the message window. The button toggles control of the Event Logger between Zeb's display manager and the X server window manager. The Events and Debug menu buttons are used primarily for development and real-time operations and should not be used for post-processing.
