Adding data to horizontal plots

The ability to overlay data is fundamental to zeb, and no more so than in the horizontal plot format. The mechanism for overlaying data is quite simple.

Adding data

Recall that all zeb configurations include an iconbar:

Most of the icons in the iconbar correspond to available sources of data. If you were interested in, say, an aircraft track, the sequence of steps to display that track in a horizontal window would be:

  1. Move to the aircraft icon in the icon bar.
  2. Pull down the menu and pick the aircraft that interests you.
  3. When the pointer changes to the crosshair shape, click on the window where you wish to see the track.
The window will then redraw with the new data incorporated. If you mistakenly select a data source that you do not wish to plot, clicking on the background (instead of on a graphic window) will cancel the operation.

Removing data

To remove a data overlay from a horizontal plot, simply move to the associated icon in the graphic window, pull down the menu with the left button, and select remove. See also disabling an overlay however.
