Start with a close configuration

One trick to making the editing process as easy as possible is to start with a configuration that is close to what you want. Most Zebra setups include a set of predefined configurations which are available; look through them to see if anything there is of interest. Remember that window sizes and placement are easily changed.

If all else fails, pull up the empty configuration (which has no graphic windows) and start from there.
