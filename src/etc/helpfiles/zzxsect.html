Zig-zag cross sections

Zig-zag cross sections take a list of platforms (usually sounding stations) and generate a vertical cross section along a polyline whose vertices are the stations. Only the two stations which bound a segment of the polyline are used to generate the data for that segment.

The main control for zig-zag cross sections is the platform selection widget, which is obtained with the right mouse button on the cross section icon. The exact appearance of this widget depends heavily on the configuration your are running, but it will be something like:

To set up a cross section:
  1. Usually you will want to clear out the existing set of platforms by using the button. You may also edit the existing list in place if you prefer.

  2. Using the platform buttons, select the group of platforms you wish to use in the cross section. Select them in the order you want them to appear in the polyline.

  3. When you have a set you are happy with, use the button to generate the new cross section.

The middle mouse button on the cross section icon may be used to bring up a field selection menu.
