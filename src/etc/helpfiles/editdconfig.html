Editing Display Configurations

A Zebra configuration will come with a set of predefined display configurations, accessible via the icon bar. However, there is no need to be limited to just the display configurations which have been provided by somebody else; the process of making custom configurations is really quite easy.

To make your own display configuration, follow the following series of steps:

  1. Start with a display configuration which is close to what you want.

  2. Add windows as desired.

  3. Delete any windows you don't want.

  4. Rearrange windows on the screen as desired.

  5. Save your configuration.

  6. Display your saved configuration at some future time.
