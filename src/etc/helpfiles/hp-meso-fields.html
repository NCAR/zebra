Displaying fields in station plots

Station plots allow the display of (usually) surface data at the observation location; they are used for mesonet stations, buoys, and related platforms. When you add a station plot to a horizontal plot window, you get something that nominally looks like this:
i.e. a bare arrow at the station location. It is possible, however, to annotate this arrow with up to four fields in the corners around the arrow. To do so, you need to go to the annotation area on the side of the graphics window:
As you move into one of the four "quadrants," it will illuminate:
You may then use the middle mouse button to pull up the field menu, getting something like this:
Select a field and release the mouse button. The display will redraw with the field annotated in the corresponding corner (and the side annotation adjusted accordingly):
...
There are two special fields which appear in the field menu:
  • None causes nothing to be plotted in that corner -- it goes blank.
  • Station causes the name of each station to be plotted in that corner.
