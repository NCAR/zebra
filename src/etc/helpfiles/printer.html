Printing hardcopy

Zebra contains a widget to ease the making of monochrome or color prints from a display window. To get the hardcopy widget, go to the global or tools icon and select Print widget. Some configurations may have this widget on a separate printer icon. In any case, the resulting widget looks like this:

Printing options

You can control how much of the screen is printed with this line:

The default setting is "one window," meaning that only a single window on the screen will be printed. If you click on the button, the widget will set up to print the entire screen instead.

The destination of your hardcopy is controlled by:

Normally you enter in the right-hand blank the name of the printer to which you wish to print. If you want to create a file on disk instead, click on the left button and enter a file name instead.

The type of printer you have should be set with:

The default is for a monochrome PostScript printer; you may change that with the menu attached to the button on the right.

By default, the output will be printed in "reverse video," meaning that black and white will be exchanged (no other colors are changed). This transformation usually produces better results when printing on white paper. You may click on:

to print things exactly as they appear on the screen.

The final line:

controls whether the widget stays on the screen after you make a print. If you only want to make one print you may wish to set this to yes.

Actually printing something

Once you have all the parameters set to your liking, there are some things to do:
  1. Make sure that the window(s) you wish to print are unobstructed. If there are parts of other windows on top, they may appear in your output (the actual behavior depends on your system).
  2. Click on the button. If you are printing a single window, the pointer will change to a crosshair, and you should click on the window to be printed.

  3. Walk to the printer and collect your output.
Clicking on Dismiss will get rid of this widget when you are finished with it.
